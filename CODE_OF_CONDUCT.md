# Code of Conduct

## Our Pledge

We aim to make this project useful, welcoming, and low-friction for people building security-sensitive software.

## Expected behavior

- Be respectful and specific.
- Focus feedback on the work, not the person.
- Assume good faith when possible.
- Keep discussions technically grounded.
- Help others reproduce and validate issues.

## Unacceptable behavior

- Harassment or personal attacks
- Doxing or threats
- Deliberate misinformation
- Repeatedly derailing technical discussion
- Abusive language in issues, pull requests, or chat

## Enforcement

Maintainers may remove or lock content that violates this code of conduct and may decline contributions that are persistently disruptive.

## Reporting

If you need to report a problem, open a private security or maintainer contact through the repository owner or GitHub contact channel.
