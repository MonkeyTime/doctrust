# DocTrust 0.1.0

This is the first public release of DocTrust.

## Included

- Signed payload specification in `spec/v1.md`
- JSON schemas for signed payloads and payment profiles
- JavaScript, Python, and .NET reference SDKs
- Local trust registry support
- Compact `dtp1z.` QR transport
- Reproducible demo flow

## Demo

Run:

```bash
cd sdk-js
node ../demo/repro.mjs
```

Expected output includes:

- a verified payload
- the issuer ID
- the document ID
- `transportPrefix` equal to `dtp1z.`
- `tamperRejected` set to `true`

## Notes

This release is intentionally small and focused on the payment-instruction verification path.
